从外部页面一键加入

Android 推荐：用 Chrome / Edge / Samsung Internet 通过 HTTPS 打开并安装 Info Dock PWA。安装后，在网页中点系统分享按钮，选择 Info Dock，即可保存到稍后看清单。

通用兜底：复制链接后回到 Info Dock，点击顶部搜索框右侧的 📝 笔记；按钮会优先读取剪贴板中的 URL 并自动保存。iOS 或不支持分享目标的浏览器建议使用这个方式，或用快捷指令打开导入链接。

桌面 / 高级备用：下面的书签工具更适合电脑浏览器书签栏；手机端不再推荐把它作为主方案。

桌面书签工具：加入 Info Dock

说明：PWA 分享目标和书签工具默认只保存页面标题、链接和来源；正文、摘要和元数据会沿用现有后台抓取能力。

📲 PWA 安装 / 分享目标

荣耀 / 华为手机建议先用浏览器菜单完成“添加到桌面 / 安装应用”，再完全关闭浏览器后重新分享网页。新版已调整 PWA 身份和分享目标配置，用于避开旧版安装残留。

  1. Edge 如果提示“已安装”但桌面没有图标，请在 Edge 的应用管理或系统应用列表里删除旧 Info Dock 后刷新本页。
  2. 夸克若不弹出原生安装框，请走浏览器菜单里的“添加到桌面 / 安装应用”，这不代表网页不支持 PWA。
  3. 华为浏览器安装后分享面板仍没有 Info Dock 时，请卸载旧图标，重新从本页安装，并重启浏览器或手机后再试。
