2. 外部保存工具PWA 安装、分享目标与桌面书签工具

📲 PWA 安装 / 分享目标

荣耀 / 华为手机上，“能安装 PWA”不等于“浏览器一定会把 PWA 注册成系统分享目标”。本版已把分享目标改为独立入口 + POST multipart/form-data，并更换 PWA 身份，避免继续沿用旧 manifest。

  1. 先删除桌面旧 Info Dock 和浏览器应用管理里的旧安装记录，再从 HTTPS 页面安装本版。
  2. 安装后请完全退出安装浏览器和外部网页 App，再重新打开目标网页测试系统分享面板。
  3. 若 Edge / 华为浏览器仍没有 Info Dock，说明该浏览器安装方式没有完成 Web Share Target 系统注册；网页代码不能直接强制写入系统分享面板。

🔖 桌面书签工具 / 导入方式

Android 推荐：用支持 Web Share Target 的浏览器通过 HTTPS 安装 Info Dock PWA。安装后，在网页中点系统分享按钮，若系统已注册分享目标，会出现 Info Dock 并保存到收件箱。

通用兜底：如果 Edge / 华为浏览器安装成功但分享面板仍没有 Info Dock，说明浏览器没有完成系统分享目标注册；复制链接后回到 Info Dock，点击顶部搜索框右侧的 📥 存入收件箱。

桌面 / 高级备用：下面的书签工具更适合电脑浏览器书签栏；手机端不再推荐把它作为主方案。

桌面书签工具：加入 Info Dock

说明：PWA 分享目标和书签工具默认只保存页面标题、链接和来源；正文、摘要和元数据会沿用现有后台抓取能力。

3. 自动化与回顾搜索自动化、基础标签与回顾提醒

🔎 搜索自动化与基础标签

自动保存搜索结果页摘要默认关闭，避免未经确认静默抓取搜索。启用后，每次搜索都会在打开搜索页后，通过当前 CORS 代理后台请求该搜索结果页，并只保存纯文本摘要用于全文搜索。

🔁 回顾提醒

待回顾默认只统计状态为收件箱/待读，且创建时间超过阈值天数的收件箱条目。应用内提醒只会在页面打开时提示，不承诺页面关闭后准时通知。

4. 高级设置CORS 代理配置与连接测试

🌐 CORS 代理

配置用于网页正文、搜索快照、B站 API 等跨域请求的 CORS 代理。

🔎 小红书搜索入口检测

使用固定中性词通过当前 CORS 代理检查网页搜索入口，不会发送搜索框中的用户当前关键词，也不会阻塞正常搜索。

开启后仅在检测缓存过期时后台刷新；关闭后不会自动检测，但不会删除上一次成功模板。“立即检测”仍可手动运行。

添加信息源

只保存主页、关注理由、查看频率和 RSS/RSSHub 地址；不会自动抓取内容。

添加搜索平台

搜索地址模板可以使用 {query} 作为关键词占位符。留空则网页搜索使用 Bing 兜底；站点域名用于 Bing site 搜索。
